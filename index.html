How it works
01 — SERVE

Run a node

Join the mesh with one command. Your GPU starts serving AI inference to anyone on the network. Free for users, always.

mesh-llm --auto
node mesh-mine.js
02 — USE

Get inference

Send prompts to the mesh. Get responses from powerful open models running on real hardware. Your wallet auto-signs a receipt for the node.

node mesh-client.js "your prompt"
03 — EARN

Mint MESH

Each signed receipt = 1 MESH minted to the node operator. On-chain, verifiable, on Ethereum. Wallet created automatically by Elytro.

node mesh-mine.js --status

The MESH token

A loyalty token for the mesh-llm network. No complex tokenomics. No monetary promises. You contribute GPU time, people use it, you earn MESH. Simple as that.

Fair launch. No pre-mine. No team allocation. No VC tokens. Every single MESH is earned by serving inference.
